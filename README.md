# Simple Interest Calculator

A lightweight, user-friendly tool designed to calculate the simple interest earned or owed based on the principal amount, annual interest rate, and time period.

---

## 📌 Features
* **Quick Calculation:** Instantly computes simple interest and the total accumulated amount.
* **Responsive Design:** Works seamlessly on desktops, tablets, and mobile devices.
* **Input Validation:** Ensures only realistic, positive numbers are processed.

---

## 🧮 How It Works

The calculator uses the standard mathematical formula for simple interest:

$$A = P(1 + rt)$$

Where:
* **$P$** = Principal amount (the initial sum of money)
* **$r$** = Annual interest rate (in decimal form, e.g., $5\% = 0.05$)
* **$t$** = Time period (in years)
